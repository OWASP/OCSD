# Module 14. Secure Configuration and Deployment

## Learning Objectives

After completing this module, developers will be able to:

- Configure web servers securely with proper access controls and file permissions
- Implement secure deployment practices for code and packages
- Remove unnecessary features, debug code, and unused dependencies before deployment
- Apply security controls for cloud-based deployment services
