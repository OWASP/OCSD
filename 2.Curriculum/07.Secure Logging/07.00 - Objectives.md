# Objectives for Module 7 - Secure Logging

## Knowledge
The student will need knowledge of the following topics:
What is secure logging?
The importnace of logging in a secure manner 
Type of logs (System , applicaiton , OS logs..)
Common logging standards (centralization , log integrity..)
Risk associated with improper logging (data leakage without identifying the source of the intrusion, insufficient logging)

## Skills
Ability to design and implement secure logging mechanisms in applications 
Ability to identify sensitive data and ensure it is propely logged
Ability to undrestand the importance of log integrity, protection, and secure storage ????
Ability to integrate with logging with monitoring system ??

## Competencies
With the above knowledge and skills, a developer will be able to implement effective and secure logging practices, ensuring that application activities are properly recorded without exposing sensitive data. This competency enables better threat detection, supports incident response, and contributes to building more secure and compliant applications.
