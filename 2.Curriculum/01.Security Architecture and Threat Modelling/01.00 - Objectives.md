# Objectives for Module 1 - Security Architecture and Threat Modelling

## Knowledge
The student will need knowledge of the following topics for Module 1:\
What is security architecture\
What is threat modelling\
What is secure design\
Why developers need to be aware of these concepts.

## Skills
Ability to understand the output of a threat model\
Ability to understand security architecture and secure design.

## Competencies
With the above knowledge points, a developer wpuld be able to understand the threats that are applicable to the software being built.
